# SuiteCRM-V8-API
A Python wrapper around the SuiteCRM V8 API
## Authorization
Currently only client_credentials authorization is supported
## Currently supports
* Get data from modules
* Get data by id
## How to run the example
```
python3 example.py <url> <client_id> <client_secret>
```
